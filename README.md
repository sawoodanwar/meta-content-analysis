## Overview

Multi-platform content analysis examining **health communication** and **misinformation** on Meta platforms (Facebook and Instagram), using topic modeling and computational text analysis in R.

## Research Objectives

- Examine health-related misinformation spread on Facebook and Instagram
- Identify key topics and narratives using STM / LDA
- Analyze engagement patterns across content types
- Compare user interaction dynamics between platforms

## Methodology

1. Data Collection via CrowdTangle / Meta Content Library
2. Preprocessing with `tidytext` and `quanteda`
3. STM topic modeling with platform/time covariates
4. Engagement analysis and misinformation coding
